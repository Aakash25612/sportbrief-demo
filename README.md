# SportBrief — Sports UGC Marketplace (Frontend Demo)

Frontend prototype for a sports-focused user-generated content marketplace connecting brands with verified sports participants.

## Three Portals

| Portal | Demo login | Features |
|--------|------------|----------|
| **Brand** | brand@sportbrief.com / demo123 | Post briefs, review applications, approve content, manage payments |
| **Creator** | creator@sportbrief.com / demo123 | Browse briefs, apply, upload deliverables, track earnings |
| **Admin** | admin@sportbrief.com / demo123 | Verify athletes, monitor platform activity |

## Platform Features

- Brand brief posting with sport niche, budget, and deliverable specs
- Verified creator profiles with sport, followers, and ratings
- Application workflow — apply, shortlist, accept
- Content delivery with revision requests and approval
- Escrow payments with milestone releases (Stripe UI mock)
- Real-time activity feed

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:5173
